# Digital Barter

A modern web application for bartering and trading items online. Built with Vue.js 3 and Bootstrap 5.

## Features

- Modern, responsive design
- Item swapping functionality
- Wish list system
- Online members tracking
- Category-based browsing

## Tech Stack

- Vue.js 3
- Vue Router
- Bootstrap 5
- SCSS
- FontAwesome

## Project setup
```
npm install
```
